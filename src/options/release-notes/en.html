v0.0.5 — 2026-06-19

v0.0.4 — 2026-06-18

  • Migrated extension from Manifest V2 to Manifest V3 — now compatible with Chrome, Edge, and Brave.
  • Replaced persistent background page with a service worker (background-sw.js).
  • Added fetch-based XMLHttpRequest shim (lib/xhrShim.js) — translation APIs work in the service worker without rewriting existing code.
  • Rewrote Bing translation response parser to use regex instead of DOMParser (unavailable in service workers).
  • Unified the two separate manifests (Firefox + Chrome) into a single MV3 manifest.json.
  • Replaced chrome.browserAction / chrome.pageAction with the unified chrome.action API.
  • Updated context menu contexts from browser_action/page_action to action.
  • Moved host_permissions out of permissions (MV3 requirement).
  • Updated web_accessible_resources to MV3 object format with matches.
  • Changed keyboard shortcuts from Ctrl+Alt+… to Alt+… — Chrome rejects Ctrl+Alt combinations.
  • Guarded matchMedia and AudioContext references for service worker safety.
  • Fixed window is not defined crash in the service worker caused by TTS code.
  • Dropped Firefox support — Chromium-only build (Chrome / Edge / Brave).
  • Added GitHub Actions release workflow — builds versioned unpacked zip and signed CRX on demand.
