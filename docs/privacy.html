TWP — Translate Web Pages MV3

Privacy Policy  ·  Last updated: June 2026

Overview

TWP - Translate Web Pages is a browser extension that translates web pages in real time using Google Translate, Microsoft Bing Translator, or Yandex Translate — whichever service you choose.

The extension does not collect, store, or transmit any personal data to us. No analytics, no tracking, no accounts.

What data leaves your browser

When you translate a page or selected text, that text is sent directly from your browser to the translation service you selected:

  • Google Translate — Google's privacy policy applies
  • Microsoft Bing Translator — Microsoft's privacy policy applies
  • Yandex Translate — Yandex's privacy policy applies

The content you translate is governed by whichever service you choose to use. We have no visibility into what those services do with translation requests — please review their respective privacy policies.

What data stays on your device

The extension saves the following locally in your browser only using the Chrome storage API:

  • Your preferred translation service (Google, Bing, or Yandex)
  • Your target translation language
  • Per-site auto-translate preferences
  • UI settings (theme, popup style, etc.)

This data never leaves your device and is never sent to us or any third party.

Permissions explained

  • storage — saves your preferences locally on your device
  • activeTab — reads the current page's text so it can be translated in place
  • contextMenus — adds right-click options to translate the page or selected text
  • webRequest — handles authentication tokens required by translation service APIs
  • webNavigation (optional) — detects navigation to apply auto-translate settings
  • Host access (<all_urls>) — required to translate any page the user visits; the extension cannot know in advance which sites you'll want to translate

No remote code

All extension code is bundled locally. No JavaScript is loaded from external servers at runtime. This extension complies with the Chrome Web Store's Manifest V3 requirements.

Children's privacy

This extension does not knowingly collect any information from anyone, including children under 13.
